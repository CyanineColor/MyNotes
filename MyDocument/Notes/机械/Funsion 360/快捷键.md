### **六、实用技巧与自定义设置**

#### **1. 切换导航预设（模仿其他软件）**

在 **首选项 → 常规** 中，可将鼠标中键行为设置为：

- **Fusion**（默认）：中键平移，Shift+中键旋转
    
- **Alias**：Shift+Alt+MMB 平移，Shift+Alt+LMB 旋转
    
- **Inventor**：F2+LMB 平移，F4+LMB 旋转
    
- **SOLIDWORKS**：中键+Ctrl 平移，中键+Alt 旋转
    

**来源**：[https://www.autodesk.com.cn/support/technical/article/caas/sfdcarticles/sfdcarticles/CHS/How-to-set-custom-navigation-commands-in-Fusion-360.html](https://www.autodesk.com.cn/support/technical/article/caas/sfdcarticles/sfdcarticles/CHS/How-to-set-custom-navigation-commands-in-Fusion-360.html), [https://blog.csdn.net/hu5566798/article/details/136144186](https://blog.csdn.net/hu5566798/article/details/136144186)

#### **2. 自定义任意命令快捷键**

- 在工具栏找到目标命令，鼠标悬停1秒
    
- 点击右侧的 **三个垂直点** → 选择 **"更改键盘快捷键"**
    
- 设置喜欢的键位组合（支持 `Shift`/`Ctrl`/`Alt`/`Command` 修饰键）
    
- 快捷键会绑定到您的 Autodesk ID，跨设备同步
    

**来源**：[http://mp.weixin.qq.com/s?__biz=Mzk0MzI2NzIyOQ==&mid=2247486447&idx=1&sn=c58f700bf49f719c73187c4a2e54b54b](http://mp.weixin.qq.com/s?__biz=Mzk0MzI2NzIyOQ==&mid=2247486447&idx=1&sn=c58f700bf49f719c73187c4a2e54b54b)

#### **3. 通过文本命令修改隐藏快捷键**

对于无UI按钮的快捷键（如"Look At"）：



```bash
# 1. 查找命令ID
toolkit.byid NavToolbar /items

# 2. 修改指定命令（如LookAtCommand）
hotkey.dialog LookAtCommand
```

**注意**：避免使用系统保留键（如 `V`、`X`），除非通过此方式重置

**来源**：[https://www.autodesk.com.cn/support/technical/article/caas/sfdcarticles/sfdcarticles/CHS/Customizing-hidden-shortcut-keys-in-Fusion-360.html](https://www.autodesk.com.cn/support/technical/article/caas/sfdcarticles/sfdcarticles/CHS/Customizing-hidden-shortcut-keys-in-Fusion-360.html)

### **记忆技巧与建议**

1. **高频优先**：熟练掌握 `E`（拉伸）、`L`（直线）、`D`（尺寸）、`F`（圆角）等高频快捷键
    
2. **视图切换**：`Ctrl + 1~8` 可快速定位标准视图，`空格键`快速调出视图立方体
    
3. **跨软件过渡**：若从 SOLIDWORKS/Alias 转来，立即在首选项中切换导航模式，降低学习成本
    
4. **悬停提示**：忘记快捷键时，将鼠标悬停在命令上，右侧会显示当前绑定的键位
