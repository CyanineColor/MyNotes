# 多肉植物种植全流程及注意事项

## 一、种植全流程

### **第一阶段：前期准备**

**1. 品种选择（关键起步）**

- **新手推荐**：景天科、番杏科的吉娃娃、虹之玉、胧月、姬胧月、冬美人等易养品种
    
- **环境匹配**：北方避免蓝石莲、毛海星等寒敏感品种；南方慎选夏季休眠品种
    
- **购买建议**：优先选择带盆发货的三年生壮苗，网购裸根苗需检查根系健康
    

**2. 工具与材料配置**

- **花盆**：必须选**带排水孔**的陶盆、瓦盆或塑料盆，直径比植株大3-5cm
    
- **基质配方**：
    
    - 通用配方：泥炭土:珍珠岩:蛭石 = **3:1:1** 或 **4:3:3**
        
    - 夏季配方：颗粒土（火山石/蛭石）占60-70%，增强排水
        
    - 可选添加：5%骨粉增强养分，或1-2cm松针铺面防虫害
        
- **消毒处理**：自配土需高温蒸汽消毒，杀灭虫卵
    

### **第二阶段：种植步骤**

**3. 根系处理（成活关键）**

- 裸根苗用**多菌灵溶液浸泡20分钟**消毒[](https://www.lvzhimi.com/zaipei/141577.html)
    
- 修剪：保留主根5-7cm，剪除腐烂/细弱须根[](https://www.lvzhimi.com/zaipei/141577.html)
    
- **晾根**：阴凉处静置**48小时**，待切口完全愈合[](https://www.lvzhimi.com/zaipei/141577.html)
    

**4. 科学定植六步法**

1. 盆底铺2cm陶粒/碎瓦片作排水层
    
2. 填入1/3基质后轻压
    
3. 用镊子夹住植株茎部定位，确保根系舒展
    
4. 分层填土至根茎交界处，**宁浅勿深**[](https://www.wevul.com/community/24091.html)
    
5. 表面铺细石子固定并增强透水性
    
6. **72小时后**首次沿盆边浇透（定根水）
    

### **第三阶段：日常养护**

**5. 浇水原则**

- **核心口诀**： **"干透浇透，宁干勿湿"**[](http://mp.weixin.qq.com/s?__biz=MzI0NDY4MDkxNQ==&mid=2247614575&idx=3&sn=36a8af73da5d43a74a22e2f0242b5027)
    
- **判断方法**：牙签插入盆土，拔出后干爽则需浇水；或观察底部叶片发皱[](http://mp.weixin.qq.com/s?__biz=MzI0NDY4MDkxNQ==&mid=2247614575&idx=3&sn=36a8af73da5d43a74a22e2f0242b5027)
    
- **季节频率**：春秋季生长季7-10天/次，夏季15-20天/次，冬季一个月/次
    
- **浇水方式**：沿盆边慢浇，**避免叶心积水**导致烂芯
    

**6. 光照管理**

- **需求**：多数品种每天需**4-6小时散射光**
    
- **夏季**：必须遮阴，拉70-80%遮阳网或移至东向/北阳台
    
- **冬季**：低于5℃时移入室内，放置光线明亮处
    

**7. 施肥与修剪**

- **施肥**：春秋生长季每月施**专用液肥**1次，稀释后使用
    
- **修剪**：及时摘除枯黄叶，促进新陈代谢
    

**8. 换盆修根**

- **周期**：每**半年至一年**换盆一次
    
- **时机**：春季清明前后最佳
    
- **操作**：脱盆后去除宿土，修剪老根，消毒后重新栽植

## 二、核心注意事项

### **⚠️ 四大生死禁忌**

表格

复制

|风险点|正确做法|错误后果|
|:--|:--|:--|
|**浇水过多**|严格控水，夏季休眠期断水|烂根、黑腐病|
|**暴晒伤叶**|夏季必须遮阴，提供散射光|叶片化水、灼伤|
|**盆底积水**|确保排水孔畅通，垫陶粒层|根系腐烂|
|**低温冻害**|低于5℃移入室内，停止浇水|冻伤、死亡|

### **🌡️ 季节性养护要点**

**春季（3-5月，黄金生长期）**

- 增加浇水频率，开始施肥
    
- 可进行叶插/分株繁殖
    

**夏季（6-8月，高危期）**

- **严格遮阴**：70-80%遮光率
    
- **通风降温**：开窗+风扇，避免40℃以上高温[](http://mp.weixin.qq.com/s?__biz=MzI0NDY4MDkxNQ==&mid=2247614575&idx=3&sn=36a8af73da5d43a74a22e2f0242b5027)
    
- **控水保命**：冬型种（如玉露）完全断水，夏型种少量浇水[](http://mp.weixin.qq.com/s?__biz=MzI0NDY4MDkxNQ==&mid=2247614575&idx=3&sn=36a8af73da5d43a74a22e2f0242b5027)
    
- **防病**：每月浇多菌灵溶液预防真菌[](http://mp.weixin.qq.com/s?__biz=MzI0NDY4MDkxNQ==&mid=2247614575&idx=3&sn=36a8af73da5d43a74a22e2f0242b5027)
    

**秋季（9-11月，恢复生长期）**

- 逐渐增加光照和水量
    
- 清除夏季枯叶，检查虫害
    

**冬季（12-2月，休眠期）**

- **温度**：保持15-28℃，低于5℃必须入室
    
- **浇水**：一个月一次或完全断水
    
- **光照**：放置在窗边接受最大光照
    

### **🐛 病虫害防治**

**常见病害**

- **黑腐病/根腐病**：立即切除腐烂部分，消毒后晾根重栽[](http://mp.weixin.qq.com/s?__biz=MzI0NDY4MDkxNQ==&mid=2247614575&idx=3&sn=36a8af73da5d43a74a22e2f0242b5027)
    
- **预防**：每月用多菌灵溶液灌根
    

**常见虫害**

- **蚧壳虫**：用酒精棉签擦拭，或埋呋虫胺（小白药）[](http://mp.weixin.qq.com/s?__biz=MzI0NDY4MDkxNQ==&mid=2247614575&idx=3&sn=36a8af73da5d43a74a22e2f0242b5027)
    
- **红蜘蛛**：增加湿度，喷洒专用杀虫剂
    
- **日常检查**：重点查看叶片背面和叶缝
    

### **✨ 新手易错点提醒**

1. **叶插繁殖**：健康叶片平放**干燥土壤**上，半月后生根发芽，期间**切勿浇水**[](https://cloud.kepuchina.cn/newSearch/imgText?id=7329172045961932800)
    
2. **配土比例**：颗粒物质不足会导致板结，**至少占30%以上**[](https://cloud.kepuchina.cn/newSearch/imgText?id=7329172045961932800)
    
3. **服盆期管理**：定植后两周内**散射光养护**，不施肥
    
4. **夏型种vs冬型种**：需区分管理，夏型种（仙人掌）夏季可少量浇水，冬型种（玉露）夏季必须断水
    

通过科学选种、规范定植和精细化季节性管理，即使是新手也能养出饱满鲜艳的多肉植物。关键在于**管住手少浇水、遮好阴挡强光、勤通风防闷热**三大原则
