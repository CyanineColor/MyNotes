方法1：使用Snap

Snap是Ubuntu的一个包管理系统，它允许你安装和管理软件包，包括.NET运行时。

    ‌安装Snap（如果尚未安装）‌


    sudo apt update
    sudo apt install snapd

    ‌安装.NET SDK‌


    sudo snap install dotnet-sdk --classic

    ‌安装.NET运行时‌

    sudo snap install dotnet-runtime-3-1 --classic  # 以3.1版

### 安装运行时

通过 ASP.NET Core 运行时，可以运行使用 .NET 开发且未提供运行时的应用。 以下命令将安装 ASP.NET Core 运行时，这是与 .NET 最兼容的运行时。 在终端中，运行以下命令：

Bash复制

```
sudo apt-get update && \
  sudo apt-get install -y aspnetcore-runtime-8.0
```

作为 ASP.NET Core 运行时的一种替代方法，你可以安装不包含 ASP.NET Core 支持的 .NET 运行时：将上一命令中的 `aspnetcore-runtime-8.0` 替换为 `dotnet-runtime-8.0`：

Bash复制

```
sudo apt-get install -y dotnet-runtime-8.0
```
